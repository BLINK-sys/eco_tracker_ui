ПЕРЕМЕННЫЕ ОКРУЖЕНИЯ ДЛЯ RENDER.COM (Frontend)

Обязательные переменные для настройки в Render Dashboard:

1. VITE_API_URL
   Значение: https://your-backend-url.onrender.com/api
   (Замените на реальный URL вашего backend API после развертывания)

2. NODE_VERSION
   Значение: 20.11.0

Примечание:
- VITE_API_URL должен указывать на ваш backend API на Render
- Не забудьте добавить /api в конце URL
- После развертывания backend обновите эту переменную на реальный URL
